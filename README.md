# X 리포스트 추첨기 (X Repost Picker)

X(구 트위터) 리포스트 이벤트 당첨자를 추첨하는 웹 애플리케이션입니다.

## 주요 기능

- **참가자 등록**: X 리포스트 목록을 복사/붙여넣기로 일괄 등록하거나 수동으로 개별 등록
- **추첨 애니메이션**: 카드가 화면에서 튕기며 돌아다니는 시각적 효과
- **당첨자 선정**: 드라마틱한 효과음과 함께 당첨자를 한 명씩 공개
- **당첨자 카드 스택**: 당첨자가 선정될 때마다 가운데에서 카드가 쌓이는 애니메이션
- **순위 기능**: 선택적으로 당첨 순위 표시 가능
- **히스토리**: 추첨 결과를 로컬 스토리지에 저장하여 조회 가능
- **포스트 기능**: 당첨 결과를 X에 포스팅하기 쉽게 복사 기능 제공 (250자 단위 자동 분할)
- **다크/라이트 모드**: 테마 전환 지원

## 기술 스택

- **Frontend**: React 19, TypeScript
- **Bundler**: Vite 7
- **Styling**: Tailwind CSS 3, DaisyUI 5
- **Storage**: LocalStorage

## 설치 및 실행

```bash

# 개발 서버 실행

## 사용 방법

### 1. 참가자 등록

#### 방법 1: 자동 추출 스크립트 (추천)
1. X에서 리포스트 목록 페이지 열기 (예: `https://x.com/username/status/123456/retweets`)
2. 브라우저 개발자 도구 열기 (F12)
3. Console 탭에서 제공된 스크립트 실행
4. 자동으로 복사된 내용 붙여넣기

#### 방법 2: 수동 입력
- X username을 직접 입력하여 개별 등록

### 2. 추첨 설정
- **이벤트명**: 추첨 이벤트 이름 설정
- **추첨 인원**: 당첨자 수 설정 (빠른 조작 버튼: +1, -1, 50%, 25%, 12.5%)
- **순위 표시**: 당첨 순위 표시 여부 선택

### 3. 추첨 시작
- "추첨 시작" 버튼 클릭
- 카드가 화면에서 빠르게 돌다가 당첨자가 한 명씩 선정됨
- 당첨자 카드가 중앙에 쌓이며 표시

### 4. 결과 활용
- **프로필 보기**: 당첨자의 X 프로필로 이동
- **포스트하기**: 당첨 결과를 X에 포스팅하기 위한 텍스트 복사
- **히스토리**: 이전 추첨 기록 조회

├── utils/

```

## 프로필 이미지

프로필 이미지는 [unavatar.io](https://unavatar.io) 서비스를 사용하여 자동으로 가져옵니다.
- 형식: `https://unavatar.io/x/{username}`
- 이미지를 불러올 수 없는 경우 기본 아바타가 표시됩니다
